## 📁 Repository 폴더 구조
```
📂 algorithm-study
 ├── 📂 week01
 │   ├── 📂 1234  # 공통 문제 
 │   │   ├── 📄 1234_홍길동.py
 │   │   ├── 📄 1234_고길동.cpp
 │   ├── 📂 5678
 │   ├── 📂 홍길동  # 개인 자유 문제
 │   │   ├── 📄 12345_홍길동.py
 │   │   ├── 📄 23456_홍길동.py
 │   ├── 📂 고길동
 ├── 📂 week02
```
`공통 문제`는 `공통 문제 폴더` 내에 파일 추가
```
week01/BOJ_문제번호/BOJ_문제번호_본인이름
```
`개인 문제`는 `본인 이름 폴더` 생성 후 파일 추가
```
week01/본인이름/BOJ_문제번호_본인이름
```
